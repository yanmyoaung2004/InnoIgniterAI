# 🔐 InnoIgniterAI

AI-powered cybersecurity platform for real-time threat detection, analysis, and intelligent response.

---

## 🚀 Overview

**InnoIgniterAI** is an advanced cybersecurity system that leverages **LLMs, multi-agent architecture, and SIEM integration** to detect and analyze online threats such as phishing links, malicious files, and email-based attacks.

The system transforms complex security data into **clear, actionable insights**, making cybersecurity accessible to both technical and non-technical users.

---

## 🧠 Key Features

* 🔍 **Phishing & Malware Detection**
  Detects malicious URLs, files, and emails in real time

* 🤖 **AI Cybersecurity Assistant**
  Context-aware Q&A using RAG (Retrieval-Augmented Generation)

* 📊 **SIEM Integration (Wazuh)**
  Log analysis, anomaly detection, and threat correlation

* 🧩 **Multi-Agent Architecture**
  Modular AI agents coordinated via LangGraph

* 🔗 **Blockchain Evidence Ledger**
  Tamper-proof storage of security events

* 🌐 **Multilingual Support**
  English & Myanmar chat + voice interface

---

## 🏗️ Architecture

* **Backend:** Python (FastAPI)
* **AI Layer:** LLMs, LangChain, LangGraph, RAG
* **Security Monitoring:** Wazuh (SIEM)
* **Storage:** Vector DB + structured logs
* **System Design:** Microservices + MCP (Main Control Program)

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yanmyoaung2004/InnoIgniterAI.git
cd InnoIgniterAI
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the server

```bash
uvicorn main:app --reload
```
